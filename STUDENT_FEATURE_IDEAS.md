# Student Feature Ideas for GMRIT Academic Calculator

## Current Features
- Internal marks calculation
- SGPA calculation  
- Results export (PDF/Excel)
- Visitor tracking system

---

## **Academic Features**

### 1. **CGPA Calculator**
- Track cumulative GPA across all semesters
- Visual progress charts
- Target CGPA goal setting
- "What-if" scenarios for future grades

### 2. **Attendance Tracker**
- Subject-wise attendance percentage
- Minimum attendance requirement warnings
- Leave calculator
- Attendance impact on grades analysis

### 3. **Exam Schedule Manager**
- Upload/download exam timetable
- Study planner with countdown timers
- Subject-wise study time allocation
- Integration with internal marks data

### 4. **Grade Predictor**
- Predict final grades based on current performance
- Target marks needed for desired grades
- Historical grade trends
- Comparison with class average (if data available)

### 5. **Backlog/Supplementary Calculator**
- Track failed subjects
- Calculate required marks for clearing backlogs
- Supplementary exam planning
- Impact on overall CGPA
